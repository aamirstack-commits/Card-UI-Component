# Card UI Component

A simple and responsive **Card UI Component** built using **HTML and CSS**.
This project demonstrates basic front-end concepts like layout, styling, hover effects, and reusable UI components.

## 📌 Features

* Clean and minimal card design
* Responsive layout using CSS
* Image section with rounded corners
* Tag capsules (Nature, Lake)
* Hover shadow effect on card
* Styled button with hover effect
* Centered layout using Flexbox

## 🛠️ Built With

* HTML5
* CSS3

## 📷 Preview

This card component includes:

* An image section
* Category tags
* Title and description
* Call-to-action button

It can be used in:

* Blog cards
* Profile cards
* Product cards
* UI component libraries
